# Novel Sorting Algorithms
### Boolean Threshold Decomposition & Weighted Median Applications

A final year B.Tech project at Mahindra University, Hyderabad.
Supervised by **Dr. Garimella Rama Murthy**.

## What is this?
A framework that converts numerical arrays into Boolean matrices
to compute median, weighted median, and order statistics —
without traditional sorting.

## Key Results
- ✅ Federated Learning accuracy: **90.51%** (best among all methods)
- ✅ ECG denoising MSE: **0.0022**, SNR: **20.53 dB**
- ✅ Faster execution: **6.05×10⁻⁵s** vs 1.07×10⁻⁴s conventional

## Applications
- Image denoising
- ECG signal denoising  
- Byzantine-robust federated learning
